Getting & Creating

  • git init --> Initializes an existing directory as a Git repository
  • git clone [url] --> retrieves an entire repository from a local via url

Stage & Snapshot

  • git status --> shows modified files in your working directory which are staged for the next commit
  • git add [file] --> adds a file as it is to your next commit (stage)
  • git reset [file] --> unstage changes to a file while retaining the changes in your working directory
  • git add [file name] --> to add a single file to GitHub
  • git diff --> see what changes have been made since you last staged
  • git commit -m "[descriptive message]" --> commit your staged conent as a new commit snapshot

Branch & Merge

  • git branch --> lists all your branches an * will indicate your current branch
  • git branch-a --> lists all branches, local and remote
  • git branch [branch-name] --> creates a new branch in the current commit
  • git checkout --> switch to another branch and check it out intor your working directory
  • git log --> show all commits in your current branches history
  • git stash --> stash changes in a dirty working directory (temporary commits)
  • git stash clear --> removes all stashed entries

Share & Update

  • git push origin [branch name] --> pushes changes in current branch to your remote repository
  • git push --> pushes changes to remote repository (remebered branch)
  • git pull --> updates local repository to the newest commit
  • git pull origin [branch name] --> pulls changes from remote repository

Inspect & Compare

  • git log --> shows the commit history of the currently active branch
  • git log --follow [file] --> show the commits that changed a file, even across file renames
  • git how [SHA] --> show any object in Git in human-readable format
  • git config --> sets automatic command line colouring for reviewing
